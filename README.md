# Barzel Analytics V3

Professional Real Estate Market Intelligence Platform

## Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui, Zustand, Recharts, Leaflet
- **Backend**: FastAPI, pandas, scikit-learn, ReportLab, Anthropic Claude

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Architecture
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API docs: http://localhost:8000/docs
